# :material-cog-outline:{ .md .middle } 设置配置

!!! tip
    请不要担心配置错误。`应用设置` 按钮旁边有一个 `重置设置` 按钮，可以将面板重置为默认设置。

!!! info
    进行任何更改后，您必须更新您的订阅。

<div class="grid cards" markdown>

- :material-cog-outline:{ .lg .middle } __VLESS - Trojan__

    ---

    配置通用的 VLESS 和 Trojan 设置，例如 DNS 服务器、纯净 IP、端口等。

    [:octicons-arrow-right-24: 开始](vless-trojan.md)

- :material-cog-outline:{ .lg .middle } __Fragment__

    ---

    配置 Fragment 设置以减轻 ISP 或区域性中断。

    [:octicons-arrow-right-24: 开始](fragment.md)

- :material-cog-outline:{ .lg .middle } __Warp__

    ---

    __Warp__ 和 __Warp PRO__ 订阅的通用 Warp 设置，包括端点、扫描器、IPv6 等。

    [:octicons-arrow-right-24: 开始](warp.md)

- :material-cog-outline:{ .lg .middle } __Warp PRO__

    ---

    旨在绕过 ISP 对 Warp 封锁的 PRO 设置。

    [:octicons-arrow-right-24: 开始](warp-pro.md)

- :material-cog-outline:{ .lg .middle } __路由规则__

    ---

    添加路由规则以绕过本地网站、拦截广告、拦截不需要的内容等。

    [:octicons-arrow-right-24: 开始](routing-rules.md)

</div>
